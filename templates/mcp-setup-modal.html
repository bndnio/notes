{{tokenSection}}
1

Save your token

Add NOTES_MCP_TOKEN to your shell profile so Claude Code can read it:

zsh (macOS default)

echo 'export NOTES_MCP_TOKEN=your_token_here' >> ~/.zshrc && source ~/.zshrc

bash

echo 'export NOTES_MCP_TOKEN=your_token_here' >> ~/.bashrc && source ~/.bashrc

Claude Code reads ${NOTES_MCP_TOKEN} from your shell environment.

2

Install the MCP server

One-liner

After setting your token above, run this in your terminal:

curl -s {{appUrl}}/install-mcp/claude-code | sh

Manual — CLI

claude mcp add --transport http notes {{appUrl}}/mcp \
  -H "Authorization: Bearer ${NOTES_MCP_TOKEN}"

Manual — .mcp.json

{
  "mcpServers": {
    "notes": {
      "type": "http",
      "url": "{{appUrl}}/mcp",
      "headers": { "Authorization": "Bearer ${NOTES_MCP_TOKEN}" }
    }
  }
}
3

Verify

Restart Claude Code and ask Claude to save a note — the save_note tool should appear and confirm it saved.
